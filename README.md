Brainf*ck Interpreter in Scala
==============================

Parser and interpreter for the language **Brainfuck**, written using
[Scala Parser Combinators][parser-comb].

For a syntax guide, please see the [Wikipedia article][wiki].

## Example programs

### Adding two values
```
[->+<]
```
